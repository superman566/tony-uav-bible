# 图像识别的数学基础


## 线性代数

### 向量（Vector）

**一句话理解**：向量 = 既有大小、又有方向的量，可以想象成一支箭，从一个点指向另一个点。

**和标量的区别：**

| | 标量（Scalar） | 向量（Vector） |
|-|----------------|-----------------|
| 有没有方向 | 没有 | 有 |
| 生活例子 | 温度、身高、体重 | 速度、位移、力 |
| 表示方式 | 一个数字，如 `85` | 一串数字，如 `(3, 4)` |

**在数学/编程里的样子**：向量本质上就是**一串按顺序排列的数字**。

```
二维向量：(3, 4)          — 平面上一个箭头
三维向量：(3, 4, 5)        — 空间中一个箭头
n 维向量：(x1, x2, ..., xn) — n 个数字排成一列，人脑无法想象，但计算机算得动
```

以 (3, 4) 为例，画在坐标系里就是从原点指向 (3,4) 的一支箭：

```
y
4 |        •(3,4)
3 |       /
2 |      /
1 |     /
0 |____/________ x
  0  1  2  3
```

**核心特点：**

1. **有大小（模/长度）**：(3,4) 的长度 = √(3² + 4²) = 5（勾股定理）
2. **有方向**：箭头指向哪，方向就是哪
3. **能相加**：首尾相接，`(1,2) + (3,1) = (4,3)`
4. **能数乘（缩放）**：乘以 2，长度翻倍、方向不变；乘以 -1，方向反转
5. **维度可以任意多**：不局限于二维/三维，784 维、1000 维的向量在计算机里很常见

> **和图像识别的关系**：一张 28×28 的灰度图，把每个像素值按顺序排成一行，就变成了一个 784 维的向量——这就是神经网络"看"图片、做计算的基本形式。

**代码示例（numpy）：**

```python
import numpy as np

v1 = np.array([3, 4])
v2 = np.array([1, 2])

v1 + v2              # 向量相加 → [4, 6]
v1 * 2               # 数乘（缩放）→ [6, 8]
np.linalg.norm(v1)   # 求模（长度）→ 5.0
```

**单位向量（Unit Vector）**

**一句话理解**：长度（模）正好等于 1 的向量，只表示"方向"，不表示"大小"。

**怎么得到**：把向量除以它自己的模，这个过程叫**归一化（normalize）**。

```
向量 (3, 4) 的模 = 5
单位向量 = (3/5, 4/5) = (0.6, 0.8)   ← 方向没变，长度被压缩成了 1
```

**为什么有用：**
- 只想表达"方向"、不想让"大小"干扰计算时用它（比如光照方向、法向量）
- 机器学习里常把特征向量归一化成单位向量，避免数值大小差异影响相似度计算

```python
v = np.array([3, 4])
unit_v = v / np.linalg.norm(v)   # → [0.6, 0.8]
np.linalg.norm(unit_v)           # → 1.0，验证模确实是 1
```

### 向量加法（Vector Addition）

**一句话理解**：两个向量相加，就是把它们对应位置的数字分别相加，得到一个新向量。

```
(1, 2) + (3, 1) = (1+3, 2+1) = (4, 3)
```

**前提条件**：两个向量的维度必须相同才能相加（二维向量不能直接加三维向量）。

**几何理解（头尾相接法）**：把向量 B 的起点接到向量 A 的终点，再从最初的起点画一条线到 B 的终点——这条线就是 A + B。

生活例子：你先往东走 3 步、往北走 1 步（向量 A）；接着再往东走 1 步、往北走 2 步（向量 B）。两段路走完，你相对起点的总位移，就是 A + B。

```
向量 A = (3, 1)：从 (0,0) 走到 (3,1)
向量 B = (1, 2)：接着从 (3,1) 走到 (3+1, 1+2) = (4,3)

最终结果：从起点 (0,0) 直接指向 (4,3) 的箭头，就是 A + B
```

**代码示例（numpy）：**

```python
import numpy as np

a = np.array([3, 1])
b = np.array([1, 2])

a + b   # 向量加法 → [4, 3]
```

> **和图像识别的关系**：深度学习里常见的"残差连接"（ResNet 的 skip connection），本质上就是把两个特征向量对应位置的数字相加——这是向量加法在图像识别模型里最直接的应用。

### 向量点积（Dot Product）

**一句话理解**：把两个向量对应位置的数字分别相乘，再把所有结果加起来，最后得到**一个数字**（不再是向量）。

```
(1, 2) · (3, 4) = 1×3 + 2×4 = 3 + 8 = 11
```

**和加法的区别**：加法算完还是一个向量，点积算完是一个标量（一个数）。

**几何意义**：点积的正负、大小，反映了两个向量"方向有多接近"。

| 两向量夹角 | 点积结果 | 例子 |
|------------|----------|------|
| 越接近 0°（方向相近） | 越大的正数 | (1,0)·(1,0) = 1 |
| 90°（垂直） | 0 | (1,0)·(0,1) = 0 |
| 越接近 180°（方向相反） | 越大的负数 | (1,0)·(-1,0) = -1 |

生活例子：两个人一起拉车。方向完全一致时，两人的力气叠加效果最好（点积大）；一人往前拉、一人往侧面拉（垂直）时，侧面那份力对车前进毫无帮助（点积为 0）；两人方向相反时，就是在互相拖后腿（点积为负）。

**代码示例（numpy）：**

```python
import numpy as np

a = np.array([1, 2])
b = np.array([3, 4])

np.dot(a, b)   # 点积 → 11
a @ b          # 等价写法 → 11
```

> **和图像识别的关系**：点积几乎是神经网络和卷积的计算核心——① 卷积运算本质就是卷积核（kernel）和图像局部区域做点积；② 神经元的输出 = 输入向量和权重向量的点积（再加偏置）；③ 比较两张脸的特征向量相似不相似（人脸识别），用的"余弦相似度"也是基于点积计算的。
