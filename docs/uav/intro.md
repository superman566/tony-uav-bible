# 无人机简介

这里是无人机学习笔记的起点，记录学习过程中的知识积累。

## 什么是无人机

无人驾驶飞行器（UAV，Unmanned Aerial Vehicle）是指不需要飞行员在机上操控的飞行器，可以远程控制或自主飞行。

## 常见分类

- **多旋翼**：四轴、六轴等，稳定性好，适合航拍和初学者
- **固定翼**：续航长、速度快，适合测绘和长距离飞行
- **直升机**：单旋翼，操控复杂，载重能力强
- **垂直起降固定翼（VTOL）**：兼具两者优点

## 核心组件

| 组件 | 作用 |
|------|------|
| 飞控（FC） | 飞行控制计算机，处理传感器数据和控制指令 |
| 电调（ESC） | 电子调速器，控制电机转速 |
| 电机 | 提供升力和推力 |
| 电池（LiPo） | 供电，影响飞行时间 |
| 遥控接收机 | 接收地面遥控器信号 |
